
# Sistema de Gerenciamento de Tarefas

Este documento representa o desenvolvimento de um sistema simples de gerenciamento de tarefas, conforme os conceitos da Engenharia de Software.

## Funcionalidades do Sistema
- Criar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas

## Descrição Geral
O sistema foi planejado para permitir o controle básico de tarefas, simulando um CRUD simples, utilizado como base para o desenvolvimento do projeto.
