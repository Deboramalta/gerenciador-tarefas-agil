# Gerenciador de Tarefas Ágil

## Descrição do Projeto
Este projeto tem como objetivo simular o desenvolvimento de um sistema simples de gerenciamento de tarefas, aplicando os conceitos da Engenharia de Software e metodologias ágeis.

## Metodologia Utilizada
A metodologia adotada foi o Kanban, utilizando o GitHub Projects para organização das tarefas nas colunas To Do, In Progress e Done.

## Funcionalidades
- Cadastro de tarefas
- Visualização de tarefas
- Edição de tarefas
- Exclusão de tarefas

## Mudança de Escopo
Durante o desenvolvimento do projeto, foi simulada uma alteração no escopo com a inclusão de uma funcionalidade adicional, representando uma nova solicitação do cliente.
